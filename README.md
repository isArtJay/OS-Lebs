# 起源

我们大三开设了操作系统原理专业课程，主要偏基础理论教学。因为我想更深入的学习这门课，就在老师讲课本的同时，又去看了**哈工大李治军老师的《操作系统》公开课程**和**北大陈向群老师的《操作系统原理》基础课程**，同时也会以**汤小丹的《计算机操作系统（第四版）》**为辅导教材。这样能保证有非常扎实的基础知识。

理论学好了，就要有实践。我就在想：“有没有一些简单的实验可以让我把之前学的理论知识给串联、实践并运用起来”，然后我带着这个思考多次 Google、Baidu ，一头扎进了搜索引擎的大海中，各种 Link 支配着我的鼠标，到最后也是不得而终。

一次偶然的机会，我在 V2EX 上发布了一个话题：[有没有好的关于操作系统的实验练习？](https://www.v2ex.com/t/725015)这个话题在 2 小时之内被点击 600 多次、20 多人收藏。终于，在话题评论区里，我看到大家分享的有关操作系统的实验项目，有清华的 ucore、南京大学的、斯坦福的，甚至还有神级课程 6.828。这使我非常的开心，也验证了那句话“只要你想学，就有东西可学”。

这个仓库就诞生于此，通过整理一些操作系统的**基础理论课程**和**优秀的实验项目**帮助像我这样的小白感受 OS 的魅力。同时，我的实验过程也会记录于此。
